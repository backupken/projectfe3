Setting Up an Account


Making a GitHub account and learning the functions will help in the long run when you need to work in teams coding for big projects or for your future position as a front end developer! GitHub is a practical resource that all coders should know how to manuever so here are a few steps teaching you how to do so.

  1. Step 1: Install Git (this is a different application from GitHub).
  2. Step 2: Install Visual Studio Code.

You're all done! There isn't much configuration that needs to be done after installing Git and VS code since this is done automatically. There are additional settings or adjustments that you can make if you would like to make any changes along the way.
