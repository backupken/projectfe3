Basic GitHub Fundamentals

In this section, we will grasp a basic understanding of GitHub's fundamental functions such as creating and cloning a repository.


How to create a repository

Creating a repository in GitHub is easy and free to do; you can create as many as you want as well. Here are some steps so you can create your first repository.

  1. Step 1: Sign into GitHub
  2. Step 2: Click on your profile icon (top-right corner of the page) then select Your repositories from the dropdown menu.
  3. Step 3: On the repositories page, click the New button (next to the "Repositories" heading).
  4. Step 4: Set the new repository to public or private
  5. Step 5 (Optional): Initialize the repository with a README file (this is recommended but not necessary).

How to clone a repository

Understanding how to clone your repository from GitHub so you can host your files on your local machine is important to know and makes the coding process faster for most. Here are a few steps on do so!

  1. Step 1: Install Git (if you haven't done so from Setting Up GitHub).
  2. Step 2: Open VS Code then open the Extensions tab.
  3. Step 3: Download the GitHub Repositories extension.
  4. Step 4: Open the extension then log in using your credentials from GitHub.
  5. Step 5: Select the repository that you want to clone to your computer
